# Enterprise Risk Register & Dashboard (Excel + Power BI)

## Overview
This project is an **Operations & Analytics** build designed to mirror an enterprise workflow for tracking operational risks and mitigation actions across a portfolio.

It includes:
- A structured **risk register**
- A standardized **mitigation action tracker**
- A dashboard for **executive-level reporting**
- Built-in logic for **overdue flags**, filtering, and trend visibility

All data is **synthetic** and created solely for portfolio/demo purposes.

---

## Key Features

### Risk Register + Action Tracker (Excel)
- Standardized fields for consistent data entry (risk category, severity, likelihood, owner, status, review date)
- Clear structure linking: **Risks → Actions → Owners → Review Dates**
- Overdue tracking using conditional logic and status fields
- Designed for filtering and audit-friendly review

### Executive Dashboard (Power BI)
- Summary of:
  - Top residual risks
  - Overdue mitigation actions
  - Risk trends over time
- Built to support decision-making and operational prioritization

---

## Why This Project
Operations teams rely on documentation, compliance tracking, and structured checklists to ensure consistency across multiple sites. This project reflects that workflow through:
- portfolio visibility  
- accountability by owner  
- audit-ready organization  
- consistent reporting
